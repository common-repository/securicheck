== Description ==

### UN OUTIL COMPLET POUR AUDITER VOTRE SITE INTERNET

Securicheck est un plugin innovant conçu pour auditer la sécurité des sites WordPress de manière approfondie et efficace. En vérifiant plus de 40 points de contrôle de sécurité, il assure une protection optimale contre les vulnérabilités potentielles. L'utilisation de Securicheck est incroyablement simple : un seul clic suffit pour lancer l'audit de sécurité. Une fois l'audit terminé, le rapport détaillé et complet est directement visible dans le tableau de bord du plugin, permettant une analyse rapide et une prise de décision éclairée.

#### Fonctionnalités :

* Lancement manuel d'audits de sécurité
* Audit de plus de 40 points de contrôle
* Analyse des connexions réussies et échouées à votre site
* Envoi de rapport détaillés par mail
* Chargement d'un ancien audit
* Suppression d'un audit
* Dissimulation de l'URL du backoffice

== Installation ==

Assurez vous que votre site est sécurisé en installant Securicheck:

1. Installez Securicheck en le trouvant dans la bibliothèque WordPress ou en téléchargeant le fichier ZIP. 
2. Activez Securicheck via le menu « Extensions » de WordPress. Securicheck est maintenant activé.
3. Accédez au menu "Réglages" de Securicheck pour définir vos préférences d'audit.
4. Accédez au menu "audit" et lancez votre premier audit.
5. Une fois votre premier audit terminé, un dashboard avec les résultats apparaîtra. Parcourez-les un par un pour vérifier la sécurité de votre site.

== Frequently Asked Questions ==

= Qu'est-ce que Securicheck ? =

Securicheck est un plugin de sécurité conçu pour auditer les sites WordPress. Il vérifie plus de 40 points de contrôle de sécurité afin de garantir une protection optimale contre les vulnérabilités et les menaces potentielles.

= Comment fonctionne l'audit de sécurité avec Securicheck ? =

L'audit de sécurité avec Securicheck est simple et rapide. Il suffit de cliquer sur un bouton pour lancer l'audit. Le plugin analyse alors le site selon ses 40 points de contrôle et génère un rapport détaillé accessible directement dans le tableau de bord.

= Comment puis-je accéder au rapport d'audit généré par Securicheck ? =

Une fois l'audit terminé, le rapport détaillé est immédiatement accessible dans le tableau de bord du plugin. Vous pouvez y consulter les résultats de l'audit, identifier les problèmes potentiels et prendre les mesures nécessaires pour sécuriser votre site.
Vous avez également la possibilité de recevoir un rapport synthétique par mail si vous le souhaitez.

= Securicheck est-il facile à installer et à utiliser ? =

Oui, Securicheck est conçu pour être facile à installer et à utiliser, même pour les utilisateurs novices. L'installation du plugin est rapide et l'interface utilisateur est intuitive, permettant de lancer des audits de sécurité en un seul clic et de consulter facilement les rapports.
